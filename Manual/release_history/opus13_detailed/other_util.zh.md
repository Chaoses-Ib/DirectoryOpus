# Directory Opus 13 - 详细版本说明

# 其它实用程序面板

- 常规：
  - 现在，右键单击面板的收起按钮（关闭旁边）后，会显示一个菜单，用于选择自动收起的面板。
  - 还可以通过配置/杂项/高级：utility_panel_autoshrink 来配置自动收起的面板。

- 日志：
  - 现在，文件日志可以通过其列进行排序。
  - 现在，您可以过滤文件和撤消日志以搜索事件。
  - 现在，您可以在脚本和 FTP 日志中搜索文本。
