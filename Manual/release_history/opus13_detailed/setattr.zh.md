# Directory Opus 13 - 详细发行说明

# 设置属性

- “现在”按钮现在是一个下拉菜单，带有“设置为当前时间”和“从文件设置”选项。
- 增加了已创建和修改时间的时移支持。使用与元数据编辑器时间字段相同的格式。
- *命令：* `SetAttr MODIFIED` 和 `CREATED` 参数现在可以获取时移字符串。
