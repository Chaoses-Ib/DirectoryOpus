# 对话框编辑器

![](/Manual/images/media/image059.png)

如果您熟悉 Visual Studio 中的对话编辑器，那么您可能会认出脚本对话编辑器，因为它在很大程度上受到 Visual Studio 原始对话编辑器的影响。

对话编辑器的四个主要部分是：

- **命令**：用于编辑对话的命令，包括定位和对齐工具。
- **工作空间**：设计对话的主要区域。
- **控件面板**：可以添加到对话框的控件。
- **属性**：编辑对话框和控件的属性。

有关如何访问对话编辑器的信息，请参阅[创建脚本对话框](../../script_dialogs/creating_script_dialogs.zh.md)。
