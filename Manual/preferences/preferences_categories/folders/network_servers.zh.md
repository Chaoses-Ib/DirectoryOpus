# 网络服务器

此页面将影响文件夹树的*网络*分支中显示的服务器。

- 你可以启用服务器缓存，这将使在执行一次后获取服务器列表的速度更快。
- 你可以指定即便 Windows 无法找到这些服务器，它们也应始终显示。
- 你可以指定永远不应显示的服务器，还可以在需要时完全禁用该树中的服务器列表。

此页面上的选项包括：

- **使用在其网络上找到的计算机填充网络文件夹**：启用此选项以启用在该树中列出网络服务器。
- **跨重启缓存计算机列表**：服务器列表将被缓存并保存至磁盘，以便在下次会话中重新加载。
- **缓存计算机列表直到关机**：服务器列表将缓存在内存中，但不会保存至磁盘。
- **不缓存计算机列表**：服务器列表将完全不缓存；Opus 将始终要求 Windows 列出服务器。
- **排除这些计算机**：你可以向此字段添加服务器名称，以将它们从该树中排除。
- **总是添加这些计算机**：你可以向此字段添加服务器名称，以使它们始终添加到该树。
