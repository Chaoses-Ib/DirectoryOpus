# 作业栏

这些选项控制可选的 [作业栏](/Manual/file_operations/copying_moving_and_deleting_files/the_jobs_bar.zh.md)，它提供了一个在后台监视和控制文件操作的方法。

- **始终显示作业栏**: 即使没有作业正在运行，作业栏也会始终显示在文件窗口的底部。
- **在新作业开始时自动显示作业栏**: 如果启用此选项，当带有进度指示器的操作打开时，作业栏会自动打开。如果您没有自动打开作业栏，您仍然可以使用 `Set JOBSBAR` 命令手动显示它。
  - **在所有文件窗口中显示**: 作业栏将显示在所有现有的文件窗口中，而不仅仅是启动该作业的文件窗口。
- **作业排队时显示箭头动画**: 如果启用了作业栏，并将进度指示器设置为 [自动最小化](../progress_indicators/README.zh.md)，则每当新的作业排队时，便会显示一个小箭头动画。
- **在作业栏按钮中显示传输速度**: 当前传输速度将显示在作业栏按钮中。
- **调整文件窗口大小以显示作业栏**: 如果启用此选项，那么在显示作业栏时将垂直调整文件窗口的尺寸以适应作业栏（并在作业栏关闭时恢复其原始尺寸）。仅当新尺寸适合屏幕时才可能这样做 - 例如，如果文件窗口窗口已最大化，则无法调整其大小，在这种情况下，作业栏将从现有窗口中占用空间。
- **作业栏上的按钮大小相同**: 打开此选项以使任务栏上的所有按钮同宽 - 如果关闭，则它们将自动调整大小以适合其内容。您还可以在像素中指定 **固定大小**。
