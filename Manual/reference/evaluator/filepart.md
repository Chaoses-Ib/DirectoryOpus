\<evalcmd\> FilePart && string && Filename component of specified path. && path && path && File or folder path. \</evalcmd\>

Returns the filename component of the specified path.

//<Example://>

    Output(FilePart("C:\one\two\file.jpg"));
    --> file.jpg
