\<evalcmd\> IsDigit 且为布尔值 && 如果 *string* 仅包含数字，则为 **True**。 && string && string && 要测试的字符串。 \</evalcmd\>

如果 *string* 完全由数字字符组成，则返回 **True**，否则返回 **False**。

//<范例://>

    Output(IsDigit("12345"))
    --> true

    Output(IsDigit("123a5"))
    --> false
