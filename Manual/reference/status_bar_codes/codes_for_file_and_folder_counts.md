# Codes for file and folder counts

The following status bar codes are used to display information about the number and size of files and folders (selected, hidden and total) in the Lister. Each code has several different forms - in a dual-display Lister this can be used to control which file display the information refers to.

| Description | Code | Details |
| --- | --- | --- |
| **Number of selected folders**  <br />Displays the number of selected folders. | {sd}  <br />{sdD}  <br />{sdL}  <br />{sdR} | Selected folders in the source file display  <br />- destination file display  <br />- left/top file display  <br />- right/bottom file display |
| **Number of selected files**  <br />Displays the number of selected files. | {sf}  <br />{sfD}  <br />{sfL}  <br />{sfR} | Selected files in the source file display  <br />- destination file display  <br />- left/top file display  <br />- right/bottom file display |
| **Number of selected items**  <br />Displays the total number of selected files and folders. | {si}  <br />{siD}  <br />{siL}  <br />{siR} | Selected items in the source file display  <br />- destination file display  <br />- left/top file display  <br />- right/bottom file display |
| **Size of all selected items**  <br />Displays the total size of all selected files (and folders, if their size has been [calculated](/Manual/basic_concepts/folder_sizes.md)).<br /><br />The size can be displayed in either bytes, or an automatic mode where the unit (bytes, KB, MB or GB) is automatically chosen depending on the size. | {sb}  <br />{sbDb}  <br />{sbLb}  <br />{sbRb}  <br />{sba}  <br />{sbDa}  <br />{sbLa}  <br />{sbRa} | Size of selected items in the source file display (in bytes)  <br />- destination file display (bytes)  <br />- left/top file display (bytes)  <br />- right/bottom file display (bytes)  <br />Size of selected items in the source file display (automatic units)  <br />- destination file display (automatic units)  <br />- left/top file display (automatic units)  <br />- right/bottom file display (automatic units) |
| **Size of all selected files**  <br />Displays the total size of all selected files (does not include selected folders). | {sbf}  <br />{sbfDb}  <br />{sbfLb}  <br />{sbfRb}  <br />{sbfa}  <br />{sbfDa}  <br />{sbfLa}  <br />{sbfRa} | Size of selected files in the source file display (in bytes)  <br />- destination file display (bytes)  <br />- left/top file display (bytes)  <br />- right/bottom file display (bytes)  <br />Size of selected files in the source file display (automatic units)  <br />- destination file display (automatic units)  <br />- left/top file display (automatic units)  <br />- right/bottom file display (automatic units) |
| **Size of all selected folders**  <br />Displays the total size of all selected folders (does not include selected files). A folder's size must have been previously [calculated](/Manual/basic_concepts/folder_sizes.md) for it to be included in this total. | {sbd}  <br />{sbdDb}  <br />{sbdLb}  <br />{sbdRb}  <br />{sbda}  <br />{sbdDa}  <br />{sbdLa}  <br />{sbdRa} | Size of selected folders in the source file display (in bytes)  <br />- destination file display (bytes)  <br />- left/top file display (bytes)  <br />- right/bottom file display bytes)  <br />Size of selected folders in the source file display (automatic units)  <br />- destination file display (automatic units)  <br />- left/top file display (automatic units)  <br />- right/bottom file display (automatic units) |
|  |  |  |
| **Total number of folders**  <br />Displays the total number of folders. | {td}  <br />{tdD}  <br />{tdL}  <br />{tdR} | Total folders in the source file display  <br />- destination file display  <br />- left/top file display  <br />- right/bottom file display |
| **Total number of files**  <br />Displays the total number of files. | {tf}  <br />{tfD}  <br />{tfL}  <br />{tfR} | Total files in the source file display  <br />- destination file display  <br />- left/top file display  <br />- right/bottom file display |
| **Total number of items**  <br />Displays the total number of files and folders. | {ti}  <br />{tiD}  <br />{tiL}  <br />{tiR} | Total items in the source file display  <br />- destination file display  <br />- left/top file display  <br />- right/bottom file display |
| **Size of all items**  <br />Displays the total size of all files (and folders, if their size has been [calculated](/Manual/basic_concepts/folder_sizes.md)).<br /><br />The size can be displayed in either bytes, or an automatic mode where the unit (bytes, KB, MB or GB) is automatically chosen depending on the size. | {tb}  <br />{tbDb}  <br />{tbLb}  <br />{tbRb}  <br />{tba}  <br />{tbDa}  <br />{tbLa}  <br />{tbRa} | Size of items in the source file display (in bytes)  <br />- destination file display (bytes)  <br />- left/top file display (bytes)  <br />- right/bottom file display (bytes)  <br />Size of items in the source file display (automatic units)  <br />- destination file display (automatic units)  <br />- left/top file display (automatic units)  <br />- right/bottom file display (automatic units) |
| **Size of all files**  <br />Displays the total size of all files (does not include folders). | {tbf}  <br />{tbfDb}  <br />{tbfLb}  <br />{tbfRb}  <br />{tbfa}  <br />{tbfDa}  <br />{tbfLa}  <br />{tbfRa} | Size of files in the source file display (in bytes)  <br />- destination file display (bytes)  <br />- left/top file display (bytes)  <br />- right/bottom file display (bytes)  <br />Size of files in the source file display (automatic units)  <br />- destination file display (automatic units)  <br />- left/top file display (automatic units)  <br />- right/bottom file display (automatic units) |
| **Size of all folders**  <br />Displays the total size of all folders (does not include files). A folder's size must have been previously [calculated](/Manual/basic_concepts/folder_sizes.md) for it to be included in this total. | {tbd}  <br />{tbdDb}  <br />{tbdLb}  <br />{tbdRb}  <br />{tbda}  <br />{tbdDa}  <br />{tbdLa}  <br />{tbdRa} | Size of folders in the source file display (in bytes)  <br />- destination file display (bytes)  <br />- left/top file display (bytes)  <br />- right/bottom file display bytes)  <br />Size of folders in the source file display (automatic units)  <br />- destination file display (automatic units)  <br />- left/top file display (automatic units)  <br />- right/bottom file display (automatic units) |
| **Number of hidden folders**  <br />Displays the number of folders that have been hidden from the file display via the various [filters](/Manual/basic_concepts/searching_and_filtering/RAEDME.md). | {hd}  <br />{hdD}  <br />{hdL}  <br />{hdR} | Hidden folders in the source file display  <br />- destination file display  <br />- left/top file display  <br />- right/bottom file display |
| **Number of hidden files**  <br />Displays the number of hidden files. | {hf}  <br />{hfD}  <br />{hfL}  <br />{hfR} | Hidden files in the source file display  <br />- destination file display  <br />- left/top file display  <br />- right/bottom file display |
| **Number of hidden items**  <br />Displays the total number of hidden files and folders. | {hi}  <br />{hiD}  <br />{hiL}  <br />{hiR} | Hidden items in the source file display  <br />- destination file display  <br />- left/top file display  <br />- right/bottom file display |
| **Size of hidden items**  <br />Displays the total size of all hidden files (and folders, if their size has been [calculated](/Manual/basic_concepts/folder_sizes.md)).<br /><br />The size can be displayed in either bytes, or an automatic mode where the unit (bytes, KB, MB or GB) is automatically chosen depending on the size. | {hb}  <br />{hbDb}  <br />{hbLb}  <br />{hbRb}  <br />{hba}  <br />{hbDa}  <br />{hbLa}  <br />{hbRa} | Size of hidden items in the source file display (in bytes)  <br />- destination file display (bytes)  <br />- left/top file display (bytes)  <br />- right/bottom file display (bytes)  <br />Size of hidden items in the source file display (automatic units)  <br />- destination file display (automatic units)  <br />- left/top file display (automatic units)  <br />- right/bottom file display (automatic units) |
| **Size of hidden files**  <br />Displays the total size of all hidden files (does not include folders). | {hbf}  <br />{hbfDb}  <br />{hbfLb}  <br />{hbfRb}  <br />{hbfa}  <br />{hbfDa}  <br />{hbfLa}  <br />{hbfRa} | Size of hidden files in the source file display (in bytes)  <br />- destination file display (bytes)  <br />- left/top file display (bytes)  <br />- right/bottom file display (bytes)  <br />Size of hidden files in the source file display (automatic units)  <br />- destination file display (automatic units)  <br />- left/top file display (automatic units)  <br />- right/bottom file display (automatic units) |
| **Size of hidden folders**  <br />Displays the total size of all hidden folders (does not include files). A folder's size must have been previously [calculated](/Manual/basic_concepts/folder_sizes.md) for it to be included in this total. | {hbd}  <br />{hbdDb}  <br />{hbdLb}  <br />{hbdRb}  <br />{hbda}  <br />{hbdDa}  <br />{hbdLa}  <br />{hbdRa} | Size of hidden folders in the source file display (in bytes)  <br />- destination file display (bytes)  <br />- left/top file display (bytes)  <br />- right/bottom file display bytes)  <br />Size of hidden folders in the source file display (automatic units)  <br />- destination file display (automatic units)  <br />- left/top file display (automatic units)  <br />- right/bottom file display (automatic units) |
|  |  |  |
| **Show Everything mode**  <br />Indicates if *[Show Everything](/Manual/basic_concepts/searching_and_filtering/show_everything.md)* mode is active. | {hse} | "Everything" if *Show Everything* mode is active.  <br />"Filtering" if *Show Everything* mode is turned off. |
| **Number of file groups**  <br />If the file display is set to group by a column this code returns the number of distinct file groups. | {grp}  <br />{grpD}  <br />{grpL}  <br />{grpR} | Number of file groups in the source file display  <br />- destination file display  <br />- left/top file display  <br />- right/bottom file display |
